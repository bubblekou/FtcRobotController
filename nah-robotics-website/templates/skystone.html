About Us

The FIRST Tech Challenge 2019-2020 season's game was Skystone. The goal of the game is to stack as much game elements, or "stones"(which look like lego bricks), onto a foundation. It is very similar to lego. To win, one's team must stack as much stones as possible while being as efficient as possible. There are also many useful mechanics that matter to a successfull game as much as following a game objective.

More on that later, I will include a link to the game manual for anyone who thinks this is interesting :)
